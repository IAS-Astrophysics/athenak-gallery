Supplemental materials including tutorials, documentation and other resources for the AthenaK astrophysical AMR-MHD code.

## Tutorials

- **[Static (magnetized) TOV neutron star](static-neutron-star)** — the recommended entry point into AthenaK's dynamical-spacetime GRMHD capabilities. It walks through building, running, and analyzing a single isolated neutron star, constructed as a solution of the Tolman–Oppenheimer–Volkoff equations and optionally seeded with a poloidal magnetic field. Because the star is stationary in exact arithmetic, drift in the central density is a direct measure of the scheme's truncation and floor errors, making it a simple but stringent test. Covers the input file in detail, tabulated (CompOSE) equations of state, and plotting the results.

- **[Binary black hole inspiral (GW150914)](gw150914)** — how to set up, run, and analyze a binary black hole simulation targeted to GW150914, the first gravitational-wave event detected by LIGO. Uses the Z4c formulation with `TwoPuncturesC` initial data and block-based AMR with puncture tracking. Covers the grid and initial-data setup, running at scale, puncture-trajectory and constraint diagnostics, apparent-horizon finding via the Einstein Toolkit, and gravitational-wave extraction by both fixed-radius integration and Cauchy-characteristic extraction (CCE) with SpECTRE.

- **[Binary neutron star merger (Kadath/FUKA)](kadath)** — how to set up and run a binary neutron star simulation using spectral initial data generated by FUKA (the Frankfurt University/Kadath framework). Covers a required thread-safety patch to Kadath's memory pools for use inside AthenaK's OpenMP regions, the two-stage compilation, generating and ingesting the FUKA initial data, and the tabulated-EOS configuration needed for the evolution.
